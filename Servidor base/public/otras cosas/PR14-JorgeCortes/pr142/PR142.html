Pivot Table Basic


Analitze and Sumarize
You can use pivot tables to analyze any set of data, quickly group and summarize values, and identity interessing patterns and trends.


Fields and Source Data
Pivot tables are based on fields which correspond to columns in separate table of source data. You choose which fields to include in the pivot table


Columns, Rows and Values
Use fields to create columns and rows in the pivot table and to specify which data is summaritze. In this example, the month columns are based on the date field and subtotals are calculated using data from the units field.


Pivot Table Options
To edit pivot table, select it, click or tab the organize buttom in the toolbar, and then choose Pivot Options. You can add, remove, and reorder fields, and choose options for how data is summarize, sorted,labeled and more.
